# Architecture Overview

This project implements a secure, scalable, and highly available 3-tier Node.js application
using Azure managed services and Infrastructure as Code.

## Key Design Goals
- High availability and fault tolerance
- Zero-downtime deployments
- Strong security boundaries between tiers
- Centralized observability
- Global content delivery
